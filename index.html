RedCode

RedCode adalah editor kode ringan bergaya VS Code dengan auto-indent, explorer folder, terminal, dan auto-save opsional - cocok untuk belajar dan produksi harian.

Install RedCode
Windows 10/11 - Installer MSI - Klik tombol untuk download

Untuk Apa RedCode?

RedCode dibuat untuk memberi pengalaman coding yang bersih, fokus, dan cepat. Cocok untuk pemula maupun developer yang butuh editor sederhana tapi tetap kuat.

Fitur Utama

Explorer Folder

Buka folder, klik folder untuk lihat file di dalamnya.

Auto-Indent

Menambahkan indent otomatis setelah tanda :.

Terminal

Terminal bawah (PowerShell) untuk perintah cepat.

Auto-Save

Opsional auto-save, atau simpan manual dengan Ctrl+S.

Cara Install

  1. Klik tombol Install RedCode di atas.
  2. Jalankan file RedCode-Setup.msi.
  3. Shortcut akan muncul di Desktop.

Jika kamu host di Vercel atau GitHub Pages, ganti link tombol Install ke URL GitHub Releases untuk file RedCode-Setup.msi.
